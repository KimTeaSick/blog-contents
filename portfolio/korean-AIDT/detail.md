## Overview

특수학급 학생들을 위한 접근성 중심 국어 디지털 교과서입니다.

## Key Features

- MediaDevice API 기반 녹음/재생 모듈 개발
- 노래/문장 진행에 맞춘 텍스트 **실시간 하이라이트**
- aria-label / aria-index 기반 접근성 강화
- 배속 조절 등 학습자 맞춤 기능 제공

## Tech Stack

React, JavaScript, TypeScript, Styled-Components
