## Overview

사용자가 만든 캐릭터와 AI 모델을 기반으로 상호작용하는 대화형 서비스입니다.

## Key Features

- styled-components → Emotion 마이그레이션 주도
- 컴포넌트 호환성 검증 및 자동 변환 스크립트 제작
- Atomic 디자인 시스템 + 모노레포 구조 개편
- default story 자동 생성 스크립트 작성
- i18next 기반 3개국 언어 지원
- Vite SSR 기능 개발

## Tech Stack

React, Typescript, Emotion, Vite
