## Overview

모델들이 디지털 프로필을 웹뷰 기반 앱에서 업로드하고, 태그·필터 기반 검색 기능을 제공하는 플랫폼입니다.

## Key Features

- 이미지·음악·배경 선택 등 프로필 업로드 플로우 제작
- 태그, 성별, 연령대 필터 및 검색어 검색 기능 구현
- GraphQL DTO 기반 데이터 조회 및 구성  
  → 백엔드 의존도 감소

## Tech Stack

React, Tailwind CSS, GraphQL
