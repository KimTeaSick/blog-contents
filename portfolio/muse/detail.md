## Overview

모델들이 디지털 프로필을 웹뷰 기반 앱에서 업로드하고, 태그·필터 기반 검색 기능을 제공하는 플랫폼입니다.

## Key Features

- **모델프로필 업로드 프로세스를 담당 개발**
  - 디지털 프로필을 등록하는 과정을 제작 사진을 업로드하고 원하는 배경, 음악 등을 선택 할 수 있게 제작.
- **태그와 필터, 검색어를 이용한 검색기능 제작**
  - 사용자가 원하는 모델을 찾기위해 검색어 뿐 아닌 연령대 필터, 성별 필터 등을 도입, 분류 별로 모델을 검색할 수 있도록 제작. 모델들이 프로필을 등록할때 태그를 동반하여 태그에 맞는 모델을 찾을 수 있도록 태그 검색 기능을 제작.
- **graphQL를 활용한 api통신**
  - 백엔드 개발이 외주로 진행되어 협업 및 변경 요청이 어려운 문제가 발생
  - 이를 해결하기 위해 GraphQL 기반의 API 통신을 도입하여, 프론트엔드에서 DTO를 직접 구성,단일 엔드포인트를 통해 필요한 데이터를 효율적으로 조회
  - 그 결과, 백엔드 변경에 대한 의존도를 최소화하고, 클라이언트 단에서 데이터 구조를 유연하게정의·제어할 수 있는 아키텍처를 구축

## Tech Stack

React, CSS, GraphQL
