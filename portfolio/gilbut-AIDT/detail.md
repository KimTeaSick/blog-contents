## Overview

학생들이 직접 알고리즘을 체험하며 학습할 수 있는 디지털 교과서 플랫폼입니다.

## Key Features

- 큐 기반 명령어 순차 실행 게임 제작
- 커스텀 훅 기반 관심사 분리 → 복잡도 감소
- 다양한 인터랙티브 UI 제작 (모달, 카드뉴스, 비디오 등)

## Tech Stack

React, TypeScript, Styled-Components
