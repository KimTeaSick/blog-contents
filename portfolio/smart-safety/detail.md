## Overview

건설현장에서 사용하는 안전 관리/교육 앱으로, 많은 체크리스트와 UI 요소들을 다루는 복잡한 모바일 환경을 개선한 프로젝트입니다.

## Key Features

- 키오스크 환경 대응 하이브리드 앱 UI 구성
- 컴포넌트 관심사 분리 → 복잡도 감소
- 불필요한 서버 요청 제거  
  → 네트워크 요청량 **30% 감소**
- 교육 항목 모달 Preloading → UX 향상
- Redux → Redux Toolkit 리팩토링  
  → 전체 코드량 **1/4 감소**

## Tech Stack

React Native, Styled-Components
