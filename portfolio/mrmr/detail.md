## Overview

음성 기반 피드를 업로드하고 상호작용하는 SNS 모바일 앱입니다.

## Key Features

- Recoil atom/selector 기반 상태관리
- Naver·Facebook 로그인 (Firebase)
- OS 정책 차이를 고려한 하이브리드 UI 구현
- UI/기능/상태 분리 구조로 클린 아키텍처 적용

## Tech Stack

React Native, Styled-Components, Recoil, Firebase
